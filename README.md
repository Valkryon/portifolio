# Portfólio

Repositório do meu portfólio profissional, desenvolvido para apresentar projetos, habilidades técnicas e identidade visual de forma clara, moderna e objetiva.

O site foi construído com foco em desempenho, organização de código e facilidade de manutenção.

---

## Tecnologias

- **React**  
- **Vite**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**

---

## Objetivo do Projeto

Este projeto tem como finalidade:

- Centralizar informações profissionais
- Exibir projetos e experiências técnicas
- Servir como base escalável para evolução contínua do portfólio
- Aplicar boas práticas de frontend moderno
